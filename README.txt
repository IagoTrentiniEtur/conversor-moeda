Projeto de um Conversor de Moedas desenvolvido em PHP para aplicar conceitos de Programação Funcional.

Funcionalidades
Conversão de moedas: Converte valores entre Dólar (USD) e Euro (EUR).
Validação: Verifica se o valor inserido é numérico e positivo.
Taxa de Câmbio: Calcula o valor convertido usando taxas de câmbio pré-definidas.

Conceitos de Programação Funcional
Funções Puras: Funções que não alteram o estado externo.
Imutabilidade: Gera um novo valor convertido sem modificar o valor inicial.
Funções de Ordem Superior: Uso de funções para validar e calcular a conversão.

Como Executar
1. Clone o repositório:
git clone https://github.com/usuario/conversor-moedas.git
cd conversor-moedas

2. Inicie o servidor PHP:
php -S localhost:8000

3. Acesse http://localhost:8000 no navegador.

Exemplo
Entrada: 100 USD para EUR
Saída: 85 EUR (com taxa de câmbio de 0,85)
